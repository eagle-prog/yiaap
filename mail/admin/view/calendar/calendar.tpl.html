Calendar Settings
Here you may configure calendar settings. You may enable/disable the calendar option in user side. Users may use the calendar feature only if the calendar option is enabled. You may also enable/disable email reminder feature for user with respect to the events created in the calendar. You may enable/disable the mouse over feature. If enabled events can be seen on mouse over along each day in the calendar.
