Inout Webmail - Upgrade to Ultimate version from Version 4.0
   
   

Note
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Whatever customizations done on the current version of Inout Webmail will be lost during the upgrade process.
So please contact our custom support team before upgradation for detailed instructions.
(** Logos as well as theme updations also may be lost **)

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1.
   Upload all files to the current Inout Webmail installation directory.
1.
   Don't replace tha attachments and userdata folder in your current Webmail Installation.
2.
   Change the access permission of the folder 'userdata' to 777 (or writable)
3.
  

Note: If you have cron job support in your server, you may create a cron job to run the file cron/birthday.php once in every day and
cron/calendar.php Once Every 5 minutes.
3.
   If you are want to update the script please go to the following URL in your browser and run it.
http://www.yourdomain.com/[SCRIPTUPLOADPATH]/install/upgrade.php.
   
   

 

Please let us know if you are facing any issue while upgrading to version 4.0.
